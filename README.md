# Pharmacy Management System

A desktop application designed to streamline pharmacy operations, featuring a robust relational database backend and a modern user interface.

### Key Features:
* **Database Management:** Built using Oracle SQL with a fully normalized schema, utilizing constraints, sequences, and views for efficient data handling.
* **User Interface:** A Python-based GUI developed with CustomTkinter for clear, intuitive interaction.
* **Core Functions:** Full control over managing medicine inventory, tracking sales transactions, and handling system records.
